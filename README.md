# TWEETER

A Social Media App

</div>

## **How to install and run locally ?**

```
$ git clone https://github.com/raunak299/clone-twitter.git
$ cd clone-twitter
$ npm install
$ npm start
```

## **Features -**

- Feed Posts
- Explore & Trending Posts
- Add , Update & Delete Post
- Add , Update & Delete Comment
- User Profile Update
- Follow / Unfollow
- Search User
- Like/unlike and Comment on Post
- Like on Comments
- Authentication:
  - User Signup
  - User Login
  - User Logout
- Fully Responsive
- share post

## **Built with -**

- React JS
- Redux Toolkit
- React Router v6
